Detailed Description

template<typename CallbackType>
class cpgf::GCallbackExtendedConnection< CallbackType >

This is an extended version of GCallbackConnection. It supports tag and auto remove features.
The connection object is used to manage callback which added to callback list.
The container returned by getDirectList() in callback list holds a list of connection object.
Use GCallbackList<Params>::ConnectionType to access the instantialized type.
