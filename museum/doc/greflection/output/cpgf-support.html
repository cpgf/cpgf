Support for cpgf library

We use the mailing list for technical support.
Feel free to subscribe and post message to the mailing list if you want to,

  • Seek for support about cpgf library
  • Give your feedback
  • Report bugs
  • Get update when new version is released
  • Discuss anything about cpgf library

Post message, send the topic to,
cpgf-user@yahoogroups.com

Subscribe, send a blank mail to,
cpgf-user-subscribe@yahoogroups.com

Unsubscribe, send a blank mail to,
cpgf-user-unsubscribe@yahoogroups.com

Note: when you subscribing to the mailing list, on Yahoo! confirmation web page, it will ask you to join the Yahoo! group.
DO NOT click it. It requires your Yahoo! account.
You should click to join the mailing list, not the group.

To read the archive messages, please visit,
http://tech.groups.yahoo.com/group/cpgf-user/messages
